Endpoints principaux

GET /api/tickets/:uniqueId

Récupère les tickets pour un écran public (pas d'authentification requise)

POST /api/auth/login

Connexion administrateur

GET /api/screens

Liste tous les écrans (authentification requise)

POST /api/screens

Créer un nouvel écran (authentification requise)

Accès aux écrans publics

Chaque écran est accessible via une URL unique :

http://localhost:3000/screen/:uniqueId
